# APGI PARAMETER SPECIFICATION TABLE

## Guide to Measurement, Typical Ranges, and Biological Grounding

---

## PART 1: CORE PARAMETERS - ACCUMULATED SIGNAL (S)

### Parameter 1a: Exteroceptive Precision ($\Pi^e$)

| Aspect | Details |
| --- | --- |
| **Definition** | Inverse variance of exteroceptive prediction error distribution: $\Pi^e = 1/\sigma_\varepsilon^e$ |
| **Units** | [1/error²] |
| **Meaning** | Confidence in sensory predictions. High $\Pi^e$ = tight, clustered errors (reliable sensory channel). Low $\Pi^e$ = scattered errors (noisy sensory channel). |
| **Biological Implementation** | **Primary:** Gain of afferent sensory pathways in thalamus/cortex (pyramidal cell input/output ratio). **Molecular:** Regulated by norepinephrine (LC-NA), acetylcholine (basal forebrain), dopamine (midbrain). **Circuit:** Thalamic reticular nucleus gates sensory relay to cortex; higher gain → higher effective Π^e. |
| **How to Measure** | **Method 1 (Behavioral threshold):** Perceptual discrimination task. Measure stimulus contrast at which subjects report 75% detection (staircase procedure). Plot psychometric curve. Slope of curve → Π^e. Steeper slope = higher Π^e. **Method 2 (Neural gain):** Single-unit recording from sensory cortex. Measure input-output tuning curve (spike response vs. stimulus intensity). Slope = neural gain ≈ Π^e. **Method 3 (Pupillometry):** Pupil diameter change to sensory stimulus. Larger dilation → higher LC-NA activity → higher Π^e. Quantify as: ΔPupil/ΔStimulus. **Method 4 (EEG power):** Spectral power of sensory-evoked responses (e.g., visual N1 component). Higher amplitude → higher gain → higher Π^e. |
| **Measurement Protocol** | **Discrimination task (30 min):** Flash visual gratings at varying contrast (100%, 50%, 25%, 12.5%, 6%). Present 20× each. Record subject detection response (yes/no). Fit logistic: P(yes) = 1/(1 + exp(-α·(contrast-threshold))). Slope α ≈ Π^e. **EEG (15 min):** Present oddball paradigm. Standard tones (1 kHz, 80% freq) vs. deviant tones (2 kHz, 20% freq). Extract mismatch negativity (MMN, 100-200 ms window). Amplitude at Fz electrode ≈ Π^e magnitude. **Pupil (10 min):** Present sudden illumination steps (100 lux increments). Record pupil dilation latency and amplitude. Rate of dilation ∝ Π^e. |
| **Typical Population Values** | Mean: 0.4-4.5 [1/(stimulus units)²]; SD: ±0.5 **By task:** Visual (low contrast): Π^e ≈ 1.0; Auditory: Π^e ≈ 0.8; Somatosensory (localization): Π^e ≈ 1.5 **By health:** Healthy controls: Π^e ≈ 1.0; Sensory deprivation/blindness: Π^e ≈ 0.3; Schizophrenia (perceptual hypersalience): Π^e ≈ 2.5 |
| **Pathological Ranges** | **Low Π^e (<0.3):** Sensory attenuation, reduced discrimination ability (autism spectrum, some PTSD); may indicate reduced LC-NA tone or damaged sensory cortex. **High Π^e (>3.0):** Perceptual hypersensitivity (anxiety, OCD, some psychosis); indicates excessive gain or noisy sensory environment. |
| **Individual Differences** | **Genetic:** ADRA2A polymorphism affects catecholamine sensitivity; AA genotype → higher Π^e. COMT Val158Met: Val carriers → higher DA → higher tonic Π^e; Met carriers → lower Π^e. **Developmental:** Early sensory deprivation (congenital cataracts, conductive hearing loss) → permanently reduced Π^e. Intensive training (musicians, athletes) → increased Π^e in trained modality. **Neurological:** Aging: Π^e declines ~10%/decade (neural noise increases). **Psychiatric:** Depression: Π^e slightly reduced (anhedonia); Anxiety: Π^e elevated (threat hypersensitivity); ADHD: Variable Π^e (inconsistent attention). |
| **How to Interpret** | **High Π^e**: "I can precisely track this stimulus; minimal noise in my perception." Adaptive for fine discrimination tasks, but costly (requires maintained neural gain). **Low Π^e**: "This stimulus is noisy or unreliable; I can't predict it well." Adaptive during sensory uncertainty but prevents learning from noisy inputs. |
| **Dependencies** | Inversely related to: sensory noise level, fatigue, attention. Positively related to: stimulus salience, task relevance, arousal (A). Modulated by: neuromodulatory state (NE, ACh, DA). |
| **Confounds to Control** | **Attention:** Filtering task by relevance (spatial, feature-based). Ask subjects to attend to specific stimulus dimension. **Arousal:** Test at consistent time of day; control sleep the night before; avoid caffeine. **Expectancy:** Use surprise paradigm (unexpected stimulus magnitudes). |

### Parameter 1b: Interoceptive Precision (Baseline) ($\Pi^i_{\text{baseline}}$)

| Aspect | Details |
| --- | --- |
| **Definition** | Inverse variance of interoceptive (bodily) prediction error distribution: $\Pi^i = 1/\sigma_\varepsilon^i$ before modulation by somatic markers. |
| **Units** | [1/error²] where error is measured in physiological units (beats/min, cmH₂O, etc.) |
| **Meaning** | Confidence in predicting one's own body state. High $\Pi^i$ = body state well-predicted (stable autonomic state, low variability). Low $\Pi^i$ = body state unpredictable (dysautonomia, high variability). |
| **Biological Implementation** | **Primary:** Insular cortex integrates vagal/afferent interoceptive signals and compares to predictions from anterior cingulate/vmPFC. Gain of this comparison = Π^i. **Molecular:** GABA, serotonin, and opioids modulate insula responsiveness. **Circuit:** Vagus nerve → nucleus tractus solitarius → dorsal insula (primary interoceptive cortex) → anterior insula (emotion integration) → vmPFC/amygdala. Precision precision = signal/noise at each stage. **State-dependent:** Parasympathetic tone increases Π^i (slow, predictable body state); sympathetic activation decreases Π^i (variable, unpredictable arousal). |
| **How to Measure** | **Method 1 (Heartbeat detection task):** Subject detects their own heartbeat by tapping in synchrony with pulse. Measure accuracy: Number correct / Number trials. Accuracy ≈ √Π^i (higher accuracy = higher precision). **Method 2 (Interoceptive sensitivity):** Heartbeat counting task. Subject counts heartbeats over 25s without taking pulse. Measure error: \|counted - actual\|. Lower error = higher Π^i. **Method 3 (Physiological variability):** Measure heart rate variability (HRV) over 5 min resting. HRV = standard deviation of inter-beat intervals. Lower HRV = more predictable heartbeat = higher Π^i. **Method 4 (HEP amplitude):** Electrocardiogram (ECG) synchronized to EEG. Measure heartbeat-evoked potential (HEP) latency and amplitude (100-300 ms post R-wave). Higher HEP amplitude → stronger interoceptive signal → higher Π^i. |
| **Measurement Protocol** | **Heartbeat detection (10 min):** Resting seated. No external stimuli. ECG electrode on chest. Subject taps finger in response to felt heartbeat. Computer times taps vs. ECG R-waves. Accuracy = #(tap within ±250ms of R-wave) / N trials. Repeat 100 trials. **Heartbeat counting (5 min):** Present time intervals (25s, 35s, 45s, randomized). Subject counts beats silently without touching pulse. At interval end, verbal report number. Compute absolute error for each trial. Mean error ≈ √(1/Π^i). **HRV (5 min):** Resting state, ECG. Extract beat-to-beat interval series. Compute RMSSD (root mean square of successive differences). RMSSD ≈ inverse Π^i (higher RMSSD = lower precision). **HEP (20 min):** Resting EEG + ECG. Segment EEG around each R-wave (±500ms). Average waveform. Measure peak-to-peak amplitude in 100-300 ms window at Cz electrode. Higher amplitude → higher Π^i. |
| **Typical Population Values** | **Heartbeat detection accuracy:** 40-70% (chance = 50% from random timing; above 60% = significantly precise). **Heartbeat counting error:** 5-20 beats/25s; σ ≈ ±3 beats. **HRV (RMSSD):** Healthy resting: 20-60 ms; σ ≈ ±15 ms. **HEP amplitude:** 0.5-3.0 μV; σ ≈ ±0.8 μV. |
| **Pathological Ranges** | **Very low Π^i (<30% detection; RMSSD >100ms; HEP <0.3μV):** Severe interoceptive insensitivity (alexithymia, some autism spectrum, dissociative disorders). **Very high Π^i (>80% detection; RMSSD <10ms; HEP >3μV):** Interoceptive hypersensitivity (anxiety, panic disorder, some PTSD). |
| **Individual Differences** | **Genetic:** BDNF Val66Met: Met allele → higher insula connectivity → higher Π^i. HTR1A polymorphism: affects serotonin sensitivity → affects insula gain. **Developmental:** Childhood adversity → reduced Π^i (insula suppression as trauma response). Athletic training (yoga, tai chi) → increased Π^i (learned attention to body). **Psychiatric:** Anxiety disorders: Π^i elevated (hypervigilant to body cues). Autism: Π^i highly variable (some high, some very low). Depression: Π^i reduced (anhedonia/interoceptive withdrawal). Schizophrenia: Π^i variable (poor body model). **Sex differences:** Females typically higher Π^i than males (~10-20% higher accuracy on heartbeat detection). |
| **How to Interpret** | **High Π^i**: "I know my body well; I can predict and interpret my internal states." Adaptive for emotion regulation (can identify feelings early), but costly (requires maintained attention to body). Correlated with emotional sensitivity, empathy, anxiety. **Low Π^i**: "My body state is opaque to me; I don't sense internal changes." Adaptive for focusing externally (less distraction from body), but costly (can't regulate dysphoria, miss health signals). Correlated with emotional numbness, alexithymia, dissociation. |
| **Dependencies** | Positively related to: parasympathetic tone (vagal activation), emotional awareness, meditation experience. Negatively related to: stress/trauma history, autonomic variability, dissociation. |
| **Confounds to Control** | **Attention:** Instruct subjects to "just feel your heartbeat" (not forced to find it). **Motivation:** Heartbeat detection task should not have incentives (removes top-down search bias). **Autonomic state:** Test at same circadian time, post-meal, consistent exercise. **Respiration:** Measure respiratory rate alongside HRV (respiratory sinus arrhythmia can inflate apparent precision). |
| **Temporal Dynamics** | Π^i varies within-session: initially high (focused attention), then may decrease (habituation) or increase (learning). Report baseline (first 3 min) separately from late-session (final 3 min) for stability estimate. |

### Parameter 1c: Somatic Bias Weight ($\beta$)

| Aspect | Details |
| --- | --- |
| **Definition** | Strength of interoceptive modulation of precision weighting. $\Pi^i_{\text{eff}} = \Pi^i_{\text{baseline}} \cdot [1 + \beta \sigma(M - M_0)]$. Dimensionless. |
| **Units** | [dimensionless]; typical range [0, 2]; can be negative (rare). |
| **Meaning** | How much does predicted homeostatic cost (somatic marker M) change the weighting of interoceptive signals? **β > 0:** Threatened body state increases interoceptive precision (safer interpretation: "My body signals are worth listening to"). **β < 0:** Threatened body state decreases interoceptive precision (avoidant interpretation: "Ignore body signals"). |
| **Biological Implementation** | **Primary mechanism:** vmPFC (ventromedial prefrontal cortex) predicts homeostatic cost (M). This prediction modulates gain of anterior insula (AI) interoceptive processing. **Molecular:** Dopamine (reward prediction error) and opioids (pain/threat valuation) tune the vmPFC→AI projection. Serotonin (via raphe nuclei) tunes insula responsiveness to threat. **Circuit:** vmPFC sends glutamatergic projections to AI; dopamine terminals at AI gate this input. High vmPFC→AI coupling (low beta) = strong threat modulation. Low coupling (high beta) = weak threat modulation. |
| **How to Measure** | **Method 1 (Anticipation task with interoceptive focus):** Subject anticipates aversive stimulus (threat cue) while monitoring heartbeat. **Condition 1:** Attend to visual cue (external). **Condition 2:** Attend to heartbeat (internal). Measure HEP amplitude in each condition. **β ≈ (HEP_threat - HEP_neutral) / HEP_neutral.** Larger difference = higher β. **Method 2 (Threat + interoceptive task):** Present threat cue (scary image, threat of shock). Simultaneously run heartbeat detection task. Compare detection accuracy during threat vs. baseline. β ≈ change in accuracy. Positive change (better detection during threat) → β > 0. **Method 3 (fMRI connectivity):** fMRI during threat anticipation task. Measure functional connectivity between vmPFC and AI. Stronger coupling → higher β. Quantify as Pearson r between vmPFC and AI timeseries. **Method 4 (Pharmacological test):** Administer beta-blocker (propranolol 40mg) to dampen peripheral signals. Test interoceptive sensitivity (heartbeat detection) with/without drug. Larger decrement with drug → higher β (more dependent on peripheral signals). |
| **Measurement Protocol** | **Anticipation task (20 min):** Present threat cue (red square) vs. safe cue (blue square) on screen. Threat cue predicts aversive stimulus (loud sound, mild shock, or disgusting image) with 75% probability. **Condition A:** Subject watches visual cue (maintain external attention). **Condition B:** Subject monitors heartbeat while viewing cue. ECG throughout. Extract HEP in 100-300ms window post R-wave during threat vs. safe cues. **β = (HEP_threat_internal - HEP_threat_external) / HEP_neutral.** Repeat 40 threat trials, 20 safe trials per condition. **fMRI threat task (15 min):** Threat prediction paradigm (red vs. blue cues predicting shock vs. safety). fMRI scans during cues. Extract vmPFC (coordinates: MNI -4, 48, -8) and AI (coordinates: MNI 36, 16, 6) BOLD timeseries. Correlate vmPFC with AI. Compute Fisher r-to-z transformation of correlation. Z-score → β estimate. |
| **Typical Population Values** | Mean: β ≈ 0.05-2.2 (moderate interoceptive modulation). SD: ±0.3. **By psychiatric group:** Healthy controls: β ≈ 0.7; Anxiety disorder: β ≈ 1.2 (elevated interoceptive coupling under threat); Depression: β ≈ 0.3 (blunted modulation); Autism spectrum: β ≈ 0.2-0.4 (weak threat-interoception coupling). |
| **Pathological Ranges** | **Very low β (<0.1):** Interoceptive avoidance; threat-related interoceptive insensitivity. Seen in: dissociation, autism spectrum, depression. **Very high β (>1.5):** Interoceptive hypervigilance; threat amplification. Seen in: anxiety disorders, panic disorder, PTSD with hyperarousal. |
| **Individual Differences** | **Genetic:** FKBP5 (stress-related plasticity gene): risk alleles → higher β in presence of threat. **Developmental:** Early adversity → initially elevated β (threat-sensitive), but chronic stress → reduced β (habituation). **Psychiatric history:** Trauma → elevated β (threat-hypersensitivity); chronic dissociation → reduced β (avoidance). **Temperament:** High neuroticism → elevated β; Low neuroticism → reduced β. |
| **How to Interpret** | **High β (>1.0):** "My body signals are especially important when I sense danger." Adaptive for threat detection (catch real dangers early), maladaptive for anxiety (over-interpret body signals as danger). **Low β (<0.3):** "My body doesn't help me assess threat." Adaptive for stress tolerance (don't amplify anxiety), maladaptive for health monitoring (miss genuine physiological changes). |
| **Dependencies** | Modulated by: threat context, arousal (A), prior trauma. Correlates with: anxiety symptoms, interoceptive awareness, heart rate reactivity. |
| **Confounds to Control** | **Peripheral physiology:** Control actual sympathetic activation (β-blockers eliminate peripheral confound; use propranolol). **Attention instructions:** Always specify whether subject should attend internally or externally. **Threat intensity:** Calibrate threat stimulus to moderate level (avoid ceiling/floor effects). |

### Parameter 1d: Noise Amplitude ($\sigma$)

| Aspect | Details |
| --- | --- |
| **Definition** | Standard deviation of sensory noise in detection task: $\sigma_{\text{noise}}$ |
| **Units** | [nats] (standard deviation) |
| **Meaning** | Amount of intrinsic neural noise in sensory processing. High σ = noisy perception (high variability, low precision). Low σ = clean perception (low variability, high precision). |
| **Biological Implementation** | **Primary:** Stochastic ion channel opening (Poisson), neurotransmitter release variability (binomial), background synaptic bombardment. **Molecular:** Ion channel kinetics (Na+, K+, Ca2+), vesicle release probability (synaptic vesicle priming). **Circuit:** Thalamic relay neurons receive noisy inputs; variability in subthreshold inputs → variability in threshold crossing. |
| **How to Measure** | **Method 1 (Trial-to-trial variability):** Detection task with repeated identical stimuli. Measure response variability across trials. Fit psychometric model with noise parameter. **Method 2 (Psychometric slope):** Fit cumulative Gaussian to detection data. σ parameter relates to noise level. **Method 3 (Neural recording):** Single-unit recording from sensory cortex. Measure spike count variability across repeated stimuli. Fano factor → σ. |
| **Measurement Protocol** | **Repeated detection (20 min):** Present identical stimulus (e.g., 50% contrast grating) 100 times. Subject reports detection (yes/no). Compute detection probability and fit psychometric. Extract σ from psychometric fit. |
| **Typical Population Values** | σ ≈ 0.15-2.5 nats. **By age:** Children: σ higher (more noise, less mature). Adults: σ moderate. Elderly: σ higher (increased neural noise). **By condition:** Rested: σ lower. Fatigued: σ higher. |
| **Pathological Ranges** | **Very low σ (<0.15):** Abnormally low noise; may indicate sensory amplification (hallucinations, psychosis). **Very high σ (>2.5):** Excessive noise; degraded perception (sensory processing disorders, autism spectrum). |
| **Individual Differences** | **Genetic:** Ion channel genes (SCN1A, CACNA1A) affect channel kinetics. **Developmental:** Synaptic pruning reduces noise. **Lifestyle:** Meditation/training may reduce σ (refined circuits). |
| **How to Interpret** | **High σ:** "My perception is noisy; I have trouble distinguishing signal from noise." Adaptive for robust processing, maladaptive for fine discrimination. **Low σ:** "My perception is clean; I can detect subtle signals." Adaptive for sensitivity, maladaptive if too low (hallucinations). |
| **Dependencies** | Inversely related to: precision (Π), signal-to-noise ratio. Modulated by: arousal, fatigue, attention. |
| **Confounds to Control** | **Stimulus consistency:** Use identical stimulus across trials. **Attention:** Maintain constant attentional state. **Fatigue:** Test at consistent time of day. |

### Parameter 1e: Composite Interoceptive Parameter ($\beta_{\Pi^i}$)

| Aspect | Details |
| --- | --- |
| **Definition** | Structurally identifiable composite parameter: $\beta_{\Pi^i} = \beta \cdot \Pi^i_{\text{baseline}}$. Combines somatic gain (β) with interoceptive precision (Π^i) to solve identifiability. |
| **Units** | [dimensionless] (product of gain × precision) |
| **Meaning** | Overall sensitivity to interoceptive signals. High β_Π^i = strong body signal processing (high gain × high precision). Low β_Π^i = weak body signal processing (low gain × low precision). |
| **Biological Implementation** | **Primary:** vmPFC→insula projection strength (β) × insula gain (Π^i). **Molecular:** Dopamine (VTA→insula) modulates gain; GABA/serotonin modulate precision. **Circuit:** vmPFC predicts homeostatic cost → modulates anterior insula gain → integrates vagal afferent signals. |
| **How to Measure** | **Method 1 (Multi-measure composite):** Combine HEP amplitude, pupil dilation, heartbeat d-prime, P3b intero/extero ratio. Use structural equation modeling to estimate β_Π^i. **Method 2 (P3b ratio):** P3b_intero / P3b_extero ≈ β_Π^i / Π^e. **Method 3 (Heartbeat detection):** d' ≈ √β_Π^i. |
| **Measurement Protocol** | **Multi-modal (90 min):** (1) Heartbeat detection (30 min), (2) Resting HEP (20 min), (3) Interoceptive oddball P3b (20 min), (4) Pupil dilation (20 min). Fit composite model. |
| **Typical Population Values** | β_Π^i ≈ 0.25-3.5. **By group:** Healthy: β_Π^i ≈ 1.0. Anxiety: β_Π^i ≈ 1.5. Depression: β_Π^i ≈ 0.7. Autism: β_Π^i variable (0.3-2.0). |
| **Pathological Ranges** | **Very low β_Π^i (<0.25):** Interoceptive insensitivity (alexithymia, dissociation). **Very high β_Π^i (>3.5):** Interoceptive hypersensitivity (anxiety, panic disorder, PTSD). |
| **Individual Differences** | **Genetic:** BDNF Val66Met affects insula connectivity. **Developmental:** Early adversity → reduced β_Π^i. **Training:** Yoga/somatic therapy → increased β_Π^i. |
| **How to Interpret** | **High β_Π^i:** "I'm very aware of my body signals." Adaptive for health monitoring, maladaptive for anxiety. **Low β_Π^i:** "I'm not very aware of my body." Adaptive for focus, maladaptive for health awareness. |
| **Dependencies** | Product of: β (somatic gain) × Π^i (interoceptive precision). Correlates with: anxiety, emotional awareness. |
| **Confounds to Control** | **Attention:** Consistent interoceptive focus. **Arousal:** Test at consistent arousal level. **Stress:** Control stress exposure. |

## PART 2: TIME CONSTANT PARAMETERS

### Parameter 2a: Signal Integration Time Constant ($\tau_S$)

| Aspect | Details |
| --- | --- |
| **Definition** | Time constant for decay of accumulated surprise signal: $dS/dt = -\tau_S^{-1} S + \text{input}$. |
| **Units** | [seconds]; typical range [0.08, 2.2] s. |
| **Meaning** | How long does accumulated evidence persist in consciousness? **Short τ_S** (100-200 ms): Fast forgetting; only immediate evidence counts (sensory-dominated, brief conscious window). **Long τ_S** (1-2 s): Slow forgetting; accumulated evidence persists (working memory-like, integrates multiple evidence). |
| **Biological Implementation** | **Primary:** Recurrent connectivity in cortical pyramidal networks. Synaptic time constants (AMPA: ~10ms, NMDA: ~100ms, GABA_B: ~100-1000ms) set the intrinsic decay. Strong recurrent excitation → longer τ_S. **From LSM (Paper 1):** Liquid state machine fading memory ≈ 500 ms. This constrains τ_S ≤ 500 ms for single-level APGI. **Neuromodulatory adjustment:** Acetylcholine (increases AMPA/NMDA ratio) → shorter τ_S (faster decay). Norepinephrine (enhances NMDA) → longer τ_S (slower decay). |
| **How to Measure** | **Method 1 (Backward masking):** Present target stimulus (T) followed by mask (M) at varying SOA (stimulus-onset asynchrony): 10, 25, 50, 100, 200, 500 ms. Measure T detection threshold vs. SOA. Plot detection accuracy as function of SOA. Decay rate → τ_S. Fit exponential: P(detect) = 1 - exp(-SOA/τ_S). **Method 2 (Attentional blink):** Rapid serial visual presentation (RSVP) of letter stream (1 item per 100ms). Present two targets (T1, T2) at different lags: T2 at lag 2, 3, 5, 8 (200, 300, 500, 800 ms after T1). Measure T2 detection as function of lag. Dip (attentional blink) centered around 200-300 ms; recovery ~500 ms. τ_S ≈ time to recover = ~500 ms. **Method 3 (EEG: P3b latency & decay):** Oddball task (standards + deviant). Extract P3b (peak ~300-600 ms). Measure amplitude decay after peak in 100-500 ms window. Exponential fit → τ_S. **Method 4 (fMRI: BOLD decay):** Event-related fMRI. Present brief stimulus (< 2 s). Measure BOLD response decay after stimulus offset. Fit exponential to BOLD signal in post-stimulus window. **Method 5 (Neural recording: spike rate decay):** Extracellular recording from prefrontal cortex. Present brief visual stimulus. Record firing rate in post-stimulus period. Fit exponential decay of firing rate. Rate constant → τ_S. |
| **Measurement Protocol** | **Backward masking (15 min):** Present target letter (T, 50 ms) at central fixation. Follow immediately with pattern mask (M, 50 ms) at variable SOA (10-500 ms in steps). Subject indicates T identity (forced choice: "T" vs "L"). 10 trials per SOA. Measure detection accuracy. Plot P(correct) vs. log(SOA). Fit exponential: P = 1 - exp(-SOA/τ_S). τ_S ≈ SOA at 63% correct (one time constant). **Attentional blink (10 min):** RSVP stream of letters (100 ms each, 20-letter stream). First target (T1) at position 3-5; second target (T2) at lag 2, 3, 5, or 8 after T1. Subject reports both identities. 20 trials per lag. Plot T2 detection vs. lag. τ_S ≈ lag at minimum (attentional blink nadir) + ~300-500 ms recovery. **EEG oddball (15 min):** 1000 ms inter-stimulus interval. Standards (80%, 1 kHz tone, 50 ms) vs. deviants (20%, 2 kHz tone). Extract P3b at Pz electrode (time window 250-600 ms). Measure amplitude decay in 100-500 ms post-peak window. Fit exponential. τ_S ≈ decay time constant. |
| **Typical Population Values** | **Behavioral (backward masking):** τ_S ≈ 50-100 ms (sensory level). **Attentional blink:** Integration window ≈ 200-500 ms (working memory level). **EEG (P3b decay):** τ_S ≈ 200-400 ms. **Resting fMRI/neurophysiology:** τ_S ≈ 500-2000 ms (slower intrinsic timescales). **Average across levels:** τ_S ≈ 0.48 s (consistent with LSM fading memory). |
| **Pathological Ranges** | **Very short τ_S (<50 ms):** Severe sensory gating deficit; target overwhelmed by mask. Seen in: early visual processing disorders, severe attention deficit. **Very long τ_S (>2000 ms):** Slow processing; difficulty disengaging from previous stimulus. Seen in: working memory overload, some depression (slow cognitive processing). |
| **Individual Differences** | **Genetic:** Normal variation in tau includes: NMDA receptor genes (NR1, NR2A, NR2B variants) → affect synaptic time constants. **Developmental:** Childhood: τ_S shorter (faster sensory processing). Aging: τ_S increases ~10%/decade (slower neural integration). **Experience:** Musicians: τ_S shorter in auditory domain (refined temporal processing). **Neurological:** Schizophrenia: τ_S shorter (reduced temporal integration) → weak P3b. Autism: τ_S variable (sometimes slower, sometimes faster). Depression: τ_S longer (slow rumination-like integration). |
| **How to Interpret** | **Short τ_S (100-200 ms):** "I only integrate evidence over brief windows; I forget quickly." Adaptive for reactive processing, maladaptive for planning (can't hold context). **Long τ_S (500-2000 ms):** "I integrate evidence over extended windows; I remember context." Adaptive for planning/learning, maladaptive for task-switching (hard to reset). |
| **Dependencies** | Related to: working memory capacity (longer τ_S → larger buffer). Modulated by: arousal (A), neuromodulatory state (ACh shortens, GABA lengthens). |
| **Confounds to Control** | **Stimulus visibility:** Ensure target is visible in baseline (no-mask) condition. **Attention:** Use exogenous cue (sudden stimulus) vs. endogenous cue (instruction) to test separate pathways. **Training:** Repeated testing can shorten apparent τ_S (practice effect). |

### Parameter 2b: Threshold Adaptation Time Constant ($\tau_\theta$)

| Aspect | Details |
| --- | --- |
| **Definition** | Time constant for relaxation of threshold toward baseline: $d\theta/dt = -\tau_\theta^{-1}(\theta - \theta_0) + \ldots$. |
| **Units** | [seconds]; typical range [0.5, 5.0] s. |
| **Meaning** | How quickly does the ignition threshold recover to baseline after a perturbation? **Short τ\_θ** (0.5-1 s): Rapid recovery; threshold resets quickly between events (enables rapid successive ignitions). **Long τ\_θ** (5-10 s): Slow recovery; threshold remains elevated after event (refractory period, prevents "attentional blink"). |
| **Biological Implementation** | **Primary:** Thalamic reticular nucleus (TRN) implements threshold gating. Intrinsic TRN bursting (T-type calcium channel kinetics) has ~1-5 s timescale. **Neuromodulatory:** Norepinephrine (LC-NA) and acetylcholine (basal forebrain) modulate TRN responsiveness. **Global mechanism:** After ignition, frontoparietal network broadcasts broadly. As this subsides (decays with τ_S), threshold has time to recover (τ_θ). If τ_θ >> τ_S, threshold stays high even as signal decays (refractory period). |
| **How to Measure** | **Method 1 (Attentional blink - recovery kinetics):** RSVP task (as in backward masking). Measure T2 detection as function of lag from T1. Plot shows dip (blink) at lag 2-4 (200-400 ms) then recovery to baseline at lag 8+ (800+ ms). Recovery curve → τ_θ. Fit exponential recovery: P(T2 detect) = 1 - A*exp(-lag/τ_θ). τ_θ ≈ lag at 63% recovery. **Method 2 (Response inhibition refractory period):** Stop-signal task with varying stop-signal delay (SSD). Measure probability of inhibition vs. SSD. Inhibition fails for short SSD (refractory), improves with longer SSD. Rate of improvement → τ_θ. **Method 3 (EEG: P3b refractory period):** Oddball with paired deviants (D1-D2 at varied intervals: 500, 1000, 2000, 5000 ms). Measure P3b amplitude to D2 as function of D1-D2 interval. Early intervals: reduced P3b (refractory). Late intervals: normal P3b (recovered). Recovery curve → τ_θ. **Method 4 (fMRI: Frontoparietal desactivation):** Event-related fMRI. Present task event (button press, decision). Measure time for frontoparietal activation to return to baseline. Return time ≈ τ_θ. Fit exponential decay of BOLD activation. |
| **Measurement Protocol** | **Attentional blink recovery (10 min):** RSVP task (as described in τ_S protocol). Specifically measure lags 2-8 with finer resolution (lag 2, 2.5, 3, 4, 5, 6, 8). Extract P(T2 detect) for each lag. Fit recovery curve. τ_θ ≈ lag at which detection returns to >90% baseline (T1-absent condition). **Stop-signal task (15 min):** Go-signal (visual cue) requiring button press. On 25% of trials, stop-signal (beep) at variable SSD (100-400 ms relative to go-signal). Subject inhibits response when stop-signal sounds. Measure probability of successful inhibition vs. SSD. Fit: P(inhibit) = 1 - exp(-SSD/τ_θ). Typical SSRT (stop-signal reaction time) ≈ 150-250 ms; relates to threshold refractory period. **EEG paired-deviant task (15 min):** Tone stream with occasional paired deviants (D1 at random interval, D2 at fixed lag: 500, 1000, 2000, 5000 ms). Extract P3b amplitude to D2 for each lag. Fit recovery: P3b(lag) = P3b_max * (1 - exp(-lag/τ_θ)). τ_θ ≈ lag at 63% recovery. |
| **Typical Population Values** | **From attentional blink:** τ_θ ≈ 200-500 ms (recovery from refractory period). **From stop-signal:** SSRT ≈ 150-300 ms (related to threshold refractory period). **From EEG recovery:** τ_θ ≈ 1-2 s (full recovery of P3b). **Average:** τ_θ ≈ 1.45 s (longer than τ_S; creates refractory period). |
| **Pathological Ranges** | **Very short τ_θ (<200 ms):** No refractory period; rapid successive ignitions (hypervigilant). Seen in: ADHD (rapid attention switching), some bipolar mania (racing thoughts). **Very long τ_θ (>5 s):** Prolonged refractory period; slow recovery (perseveration). Seen in: depression (slow cognitive flexibility), autism (difficulty task-switching). |
| **Individual Differences** | **Genetic:** Catecholamine genes (COMT, MAOA, DAT) → affect neurotransmitter levels → modulate TRN dynamics. **Developmental:** Childhood (immature TRN) → longer τ_θ (slower task switching). Adolescence → τ_θ shortens (improved flexibility). Aging → τ_θ lengthens again (~5% increase/decade). **Psychiatric:** ADHD: τ_θ short (difficulty sustaining attention). Depression: τ_θ long (difficulty disengaging). Anxiety: τ_θ normal or short (rapid threat processing). Autism: τ_θ variable. **Training:** Meditation/mindfulness → shorter τ_θ (improved attentional flexibility). |
| **How to Interpret** | **Short τ_θ (0.5-1 s):** "I recover quickly from attention-demanding events; I can switch tasks rapidly." Adaptive for multitasking, maladaptive for task persistence (attention wanders). **Long τ_θ (2-5 s):** "I stay focused on one thing; I take time to reset." Adaptive for sustained attention, maladaptive for attention-switching (get "stuck"). |
| **Dependencies** | Related to: working memory refractory period, attentional blink duration, task-switching cost. Modulated by: arousal (A), practice (reduces with training). |
| **Confounds to Control** | **Stimulus salience:** Ensure T1 and T2 are equally salient (controls for base-rate effects). **Attention instruction:** Always specify whether subject should attend to T1 only or both T1 & T2. **Circadian timing:** Test at consistent time of day (arousal state affects τ_θ). |

### Parameter 2c: Somatic Marker Time Constant ($\tau_M$)

| Aspect | Details |
| --- | --- |
| **Definition** | Time constant for vmPFC to track predicted homeostatic cost: $dM/dt = -\tau_M^{-1}(M - M^*(\varepsilon^i)) + \ldots$. |
| **Units** | [seconds]; typical range [0.5, 5.0] s. |
| **Meaning** | How quickly does vmPFC update its prediction of homeostatic cost when body state changes? **Short τ_M** (0.5-1 s): Rapid response (detect body state changes quickly). **Long τ_M** (2-5 s): Slow response (integrates body state over longer period, smooths fluctuations). |
| **Biological Implementation** | **Primary:** vmPFC (ventromedial prefrontal cortex, coordinates: MNI -4, 52, -8) receives slow insular inputs (~1-2 s latency from vagal signals). Recurrent connectivity within vmPFC integrates across time. **Circuit:** Anterior insula → vmPFC (represents bodily state); vmPFC → posterior insula/amygdala (predicts cost). Interoceptive learning strengthens vmPFC representations. **Neuromodulatory:** Serotonin (raphe→vmPFC) modulates valuation (cost prediction). Dopamine (VTA→vmPFC) signals prediction errors. Opioids (local vmPFC) gate emotional processing. |
| **How to Measure** | **Method 1 (fMRI interoceptive prediction task):** Subject anticipates threat (red square predicting shock) vs. safety (blue square). Measure vmPFC BOLD response latency and rise time. Shorter latency/faster rise → shorter τ_M. Fit exponential rise: BOLD(t) = BOLD_max * (1 - exp(-t/τ_M)). **Method 2 (Somatic marker hypothesis task):** Gambling task (Iowa Gambling Task). Measure skin conductance response (SCR) when selecting from "bad" decks (high immediate reward, high long-term loss). SCR precedes conscious awareness in healthy subjects (~1-2 s) but is delayed in vmPFC lesions or damage. Response latency → τ_M. **Method 3 (Anticipatory physiology):** Subject anticipates aversive stimulus (threat cue→shock). Measure heart rate (HR) or respiration onset time relative to cue. Heart rate acceleration begins ~500-2000 ms after threat cue (anticipatory) vs. reactive (starts at shock). Onset time → τ_M. **Method 4 (EEG: Stimulus-preceding negativity [SPN]):** Precedes predicted stimulus (cue → shock, at variable cue-shock interval: 1, 2, 4 s). Extract SPN (slow negative potential 1-4 s post-cue, measured at Cz). SPN amplitude grows gradually, reflecting vmPFC anticipatory process. Time to peak → τ_M. |
| **Measurement Protocol** | **fMRI prediction task (20 min):** Anticipation paradigm. Red square (threat cue, 75% shock) vs. blue square (safety cue, 0% shock) displayed for 3-4 s (anticipation period), then outcome (shock or safety). fMRI throughout. Extract vmPFC BOLD timeseries (3mm sphere at -4, 52, -8). Measure time to peak BOLD during anticipation (relative to cue onset). Peak time ≈ τ_M. Repeat 40 threat trials, 20 safety trials. **Somatic marker task (15 min):** Iowa Gambling Task (or simplified version). Choose cards from 4 decks; each reveals reward/loss. Decks A-B: high immediate reward ($100) but large losses→net loss. Decks C-D: lower reward ($50) but small losses→net gain. Simultaneously measure skin conductance (SCR via electrodes on fingertips). Extract SCR peaks when selecting from bad decks (anticipatory SCR) vs. neutral choices. Timing of SCR relative to choice → τ_M. **EEG SPN task (15 min):** Cue predicts outcome (shock or safety) at variable delay (1, 2, 4 s). Extract SPN as slow negativity in -4s to 0s window before outcome. Measure SPN amplitude growth rate. τ_M ≈ time to 63% of maximum SPN (one time constant). |
| **Typical Population Values** | **From fMRI:** vmPFC peak latency ≈ 1-2 s from cue onset; τ_M ≈ 1.52 s. **From somatic marker task:** Anticipatory SCR onset ≈ 500-2000 ms before shock. **From EEG SPN:** SPN peak ≈ 1-2 s pre-outcome; τ_M ≈ 0.5-1.5 s. **Average:** τ_M ≈ 1.52 s (slower than τ_S and τ_θ, consistent with vmPFC integration of visceral signals). |
| **Pathological Ranges** | **Very short τ_M (<0.3 s):** Hyperactive threat anticipation; jump to worst-case scenario instantly. Seen in: PTSD, anxiety disorder. **Very long τ_M (>5 s):** Slow threat anticipation; difficulty predicting harm. Seen in: vmPFC lesions, psychopathy (reduced amygdala-vmPFC connectivity), some autism spectrum. |
| **Individual Differences** | **Genetic:** BDNF Val66Met: Met carriers → stronger vmPFC-insula connectivity → faster τ_M. **Developmental:** Early adversity (abuse, neglect) → altered vmPFC development → changed τ_M (often longer, blunted anticipation). **Psychiatric:** PTSD: τ_M short (hyperactive anticipation). Depression: τ_M variable. Anxiety: τ_M short. Psychopathy: τ_M very long (poor anticipatory fear). **Experience:** Trauma-informed therapy → normalizes τ_M. Meditation/mindfulness → may lengthen τ_M (less reactive). |
| **How to Interpret** | **Short τ_M (0.5-1 s):** "I anticipate body consequences quickly; I'm sensitive to threat cues." Adaptive for threat avoidance, maladaptive for false alarms (over-anticipate harm). **Long τ_M (2-5 s):** "I take time to anticipate consequences; I'm less reactive." Adaptive for measured decision-making, maladaptive for threat response (too slow). |
| **Dependencies** | Correlated with: amygdala reactivity, anxiety symptoms, heart rate reactivity. Related to: vmPFC-insula connectivity strength. |
| **Confounds to Control** | **Prior threat learning:** Use novel cue-outcome contingencies (no prior experience). **Attention:** Explicitly instruct attention (attend to cue vs. ignore). **Arousal:** Test at consistent circadian time. **Medication:** Some anxiolytics/antidepressants alter vmPFC response time; document drug use. |

### Parameter 2d: Arousal Time Constant ($\tau_A$)

| Aspect | Details |
| --- | --- |
| **Definition** | Time constant for arousal state (A) to track target arousal (A_target): $dA/dt = -\tau_A^{-1}(A - A_{\text{target}}) + \ldots$. |
| **Units** | [seconds]; typical range [0.1, 30] s (bimodal: fast phasic ~0.1-0.5 s; slow tonic ~10-30 s). |
| **Meaning** | How quickly does global alertness respond to stimuli/state changes? **Short τ_A (fast, 0.1-0.5 s):** Phasic component; rapid pupil dilation, heart rate spike to sudden stimuli (reactive arousal). **Long τ_A (slow, 10-30 s):** Tonic component; slow drift in background alertness over minutes (homeostatic baseline shift). |
| **Biological Implementation** | **Fast component (τ_A^phasic ≈ 0.1-0.5 s):** Locus coeruleus (LC) phasic firing bursts to novel/salient stimuli. GABA_A receptor kinetics (~10-100 ms). **Slow component (τ_A^tonic ≈ 10-30 s):** LC tonic firing sets baseline tone; influenced by sleep-wake cycle, metabolic state, learning. Orexin/hypocretin (10-30 s) and adenosine (hours) neuromodulators. **Global implementation:** LC projects to all cortical regions (ascending reticular activating system). Output modulates gain of all pyramidal cells (increases baseline excitability). |
| **How to Measure** | **Method 1 (Pupil dilation - phasic):** Present novel stimulus (sudden sound, light flash, novel image). Measure pupil diameter change. Pupil dilation latency ≈ 150-300 ms; amplitude ≈ 0.5-2 mm. Rise time → τ_A^phasic. Fit exponential: Pupil(t) = Pupil_0 + ΔPupil*(1 - exp(-t/τ_A)). τ_A ≈ rise time to 63%. **Method 2 (Pupil dilation - tonic):** Track pupil diameter continuously over 30 min resting state (no specific stimuli). Measure slow drift in baseline. τ_A^tonic ≈ time for baseline to shift by ~5-10%. **Method 3 (P3b amplitude - arousal modulation):** Oddball task at different times of day (morning vs. evening, rested vs. fatigued). Measure P3b amplitude (marker of attentional engagement). Higher amplitude = higher arousal. Plot P3b vs. circadian phase. Circadian modulation period ≈ 24 h = τ_A^circadian. **Method 4 (EEG spectral power - arousal):** Resting EEG. Measure alpha power (8-12 Hz) as marker of arousal (high alpha = low arousal/drowsy; low alpha = high arousal/alert). Track alpha power over 30 min. Decay rate of alpha after stimulus → τ_A. **Method 5 (LC fiber photometry - direct):** In animal models, measure LC population firing. Sudden stimulus causes phasic burst (τ_A^phasic ≈ 100 ms). Baseline drift occurs over 10-30 s (τ_A^tonic). |
| **Measurement Protocol** | **Pupil dilation (phasic; 15 min):** Subject seated in dim room. Measure pupil diameter with infrared eye tracker (30 Hz sampling). Present 40 novel acoustic stimuli (100 dB, 100 ms white noise bursts) at 30s intervals. Extract pupil response in 0-1 s window post-stimulus. Fit exponential rise. τ_A^phasic ≈ time to peak (typically 100-300 ms); rise time constant ≈ τ_A. **Pupil dilation (tonic; 30 min):** Continuous pupil tracking while subject rests (reads, watches neutral video, or sits quietly). Extract baseline pupil diameter (mean over 60s windows). Track baseline over 30 min. Fit slow drift: Baseline(t) = Baseline_0 + (Baseline_∞ - Baseline_0)*(1 - exp(-t/τ_A)). τ_A^tonic ≈ time to 63% drift (typically 5-15 min). **Circadian P3b (5 sessions):** Oddball task (20 min) at times: 8 AM, 12 PM, 4 PM, 8 PM, 12 AM. Extract mean P3b amplitude for each session. Plot P3b vs. time of day. τ_A^circadian = 24 h (by definition, circadian period). Peak of P3b ≈ 10 AM (peak human alertness). **EEG alpha (30 min):** Resting EEG at Oz electrode. Compute power in 8-12 Hz band (alpha). Plot alpha over 30 min. Rising alpha (increasing drowsiness) → tau_A ≈ time constant of alpha rise. |
| **Typical Population Values** | **Phasic (pupil rise):** τ_A^phasic ≈ 0.1-0.3 s (100-300 ms); amplitude ≈ 0.5-1.5 mm. **Tonic (slow baseline drift):** τ_A^tonic ≈ 5-30 min (300-1800 s); drift rate ≈ 5-20% baseline shift over 30 min. **Circadian:** τ_A^circadian ≈ 24 h (±1 h individual variation); peak alertness ≈ 10 AM. **Overall behavioral correlate:** Reaction time varies ~5-10% within day (slower at nadir ≈ 3-4 AM, faster at peak ≈ 10 AM). |
| **Pathological Ranges** | **Absent phasic response (τ_A^phasic undefined, no pupil dilation):** Severe hypoarousal (coma, anesthesia, deep depression). **Delayed phasic response (τ_A^phasic > 1 s):** Sluggish arousal (narcolepsy, severe sleep deprivation). **Sustained/excessive phasic response (τ_A^phasic <50 ms):** Hyperarousal (acute stress, PTSD, bipolar mania). **Absent tonic modulation (flat baseline):** Autonomic failure (e.g., pure autonomic failure, some Parkinsonian syndromes). **Flat circadian rhythm (absent τ_A^circadian modulation):** Circadian rhythm disorder (jet lag, shift work disorder, dementia); may indicate loss of suprachiasmatic nucleus function. |
| **Individual Differences** | **Genetic:** PER3 gene (circadian clock): different alleles → different τ_A^circadian phase. Carriers of longer VNTR → later chronotype (peak arousal later in day). **Developmental:** Children: τ_A^phasic smaller (reactive), τ_A^circadian earlier (earlier bedtime). Adolescents: τ_A^circadian phase-delayed (~2 h later than adults). Aging: τ_A^circadian amplitude decreases (~50% by age 80); τ_A^tonic increases (slower adaptation, more lability). **Lifestyle:** Shift workers: τ_A^circadian becomes irregular/bimodal. Athletes: τ_A^phasic shorter (faster arousal responses from training). **Psychiatric:** ADHD: τ_A^phasic shorter (hyperarousal); τ_A^tonic abnormal (non-stationary baseline). Depression: τ_A^phasic may be normal, but tonic baseline shifted lower (global hypoarousal). Bipolar mania: τ_A^phasic exaggerated; τ_A^tonic elevated. Sleep disorders: τ_A^phasic/tonic all disrupted. |
| **How to Interpret** | **Short τ_A^phasic (100-200 ms):** "I react quickly to novel stimuli; I'm highly reactive." Adaptive for threat detection, maladaptive for ignoring noise (hypervigilant). **Long τ_A^phasic (500-1000 ms):** "I take time to notice novel stimuli; I'm slow to react." Adaptive for sustained attention, maladaptive for threat response. **High τ_A^circadian amplitude:** "My alertness varies a lot throughout the day." Typical for morning people or night owls. **Low τ_A^circadian amplitude:** "My alertness is stable throughout the day." Typical for introverts, deep depressives, some neurodevelopmental conditions. |
| **Dependencies** | Modulated by: sleep/wake state, circadian phase, caffeine, alcohol, stimulant drugs, stress. Correlated with: heart rate, respiration, cortisol, temperature. |
| **Confounds to Control** | **Circadian phase:** Test at same time of day or statistically control for circadian effects. **Sleep deprivation:** Ensure adequate sleep night before (~8 h). **Caffeine:** Caffeine increases τ_A amplitude (wider fluctuations); ban caffeine 4 hours before test. **Stress:** Acute stress increases τ_A^phasic amplitude; measure baseline before/after stressor. **Medications:** Stimulants (amphetamine, methylphenidate) increase τ_A^phasic. Sedatives/anxiolytics decrease τ_A^phasic. Document all medications. |

### Parameter 2e: Surprise Decay Constant ($\tau$)

| Aspect | Details |
| --- | --- |
| **Definition** | Time constant for decay of accumulated surprise signal: $dS/dt = -\tau^{-1} S + \text{input}$. |
| **Units** | [seconds]; typical range [0.15, 0.30] s |
| **Meaning** | How quickly does accumulated surprise decay? Short τ (150-200 ms): Fast forgetting; brief conscious window. Long τ (250-300 ms): Slower forgetting; extended conscious access. |
| **Biological Implementation** | **Primary:** Recurrent excitation in pyramidal networks. NMDA receptor kinetics (~100ms dominant). **Molecular:** NMDA receptor subunit composition (NR2A vs NR2B) affects decay time. **Circuit:** Frontoparietal network broadcasts; decay rate set by recurrent connectivity strength. |
| **How to Measure** | **Method 1 (Backward masking):** Target + mask at varying SOA. Detection decay → τ. **Method 2 (Attentional blink):** T2 detection vs. lag from T1. Recovery rate → τ. **Method 3 (P3b decay):** P3b amplitude decay after peak. Exponential fit → τ. |
| **Measurement Protocol** | **Backward masking (15 min):** Target (50ms) + mask at SOAs 10-500ms. Measure P(detect) vs. SOA. Fit exponential: P = 1 - exp(-SOA/τ). τ ≈ SOA at 63% correct. |
| **Typical Population Values** | τ ≈ 0.195 s (195 ms). **By age:** Children: τ shorter (faster decay). Adults: τ moderate. Elderly: τ longer (slower decay). **By task:** Visual: τ ≈ 0.18 s. Auditory: τ ≈ 0.22 s. |
| **Pathological Ranges** | **Very short τ (<0.15 s):** Rapid forgetting; poor working memory (ADHD, some autism). **Very long τ (>0.30 s):** Slow forgetting; rumination-like processing (depression, anxiety). |
| **Individual Differences** | **Genetic:** NMDA receptor variants affect decay kinetics. **Developmental:** Synaptic maturation changes τ. **Lifestyle:** Meditation may increase τ. |
| **How to Interpret** | **Short τ:** "I forget evidence quickly; only immediate signals matter." Adaptive for reactive processing. **Long τ:** "I hold onto evidence longer; can integrate across time." Adaptive for planning. |
| **Dependencies** | Related to: working memory capacity, temporal integration. Modulated by: arousal, neuromodulators. |
| **Confounds to Control** | **Stimulus timing:** Precise SOA control. **Attention:** Maintain constant attention. **Fatigue:** Test at consistent time. |

## PART 3: THRESHOLD PARAMETERS

### Parameter 3a: Baseline Threshold ($\theta_0$) and Arousal Modulation

| Aspect | Details |
| --- | --- |
| **Definition** | Baseline ignition threshold (resting state); varies as function of arousal (A) via: $\theta_0(A) = \theta_0^{\text{sleep}} + (1 - A)(\theta_0^{\text{alert}} - \theta_0^{\text{sleep}})$. |
| **Units** | [nats] = [dimensionless]; typical range [1.0, 5.5] nats. |
| **Meaning** | How much evidence (surprise) is needed to trigger conscious access at different arousal levels? **Low arousal (sleeping):** $\theta_0^{\text{sleep}}$ low (easy to ignite for survival threats). **High arousal (alert vigilance):** $\theta_0^{\text{alert}}$ high (conservative broadcast, filter noise). |
| **Biological Implementation** | **Primary:** Thalamic reticular nucleus (TRN) sets threshold gain. Hyperpolarized TRN cells → high threshold (tonic, inhibits relay). Depolarized TRN cells → low threshold (phasic, allows relay). **Arousal control:** Locus coeruleus (LC, norepinephrine) depolarizes TRN → lowers threshold during wakefulness. Cholinergic basal forebrain and dorsal raphe (serotonin) modulate. During sleep, LC silent → TRN hyperpolarizes → threshold rises (only strong stimuli pass). **Circuit:** TRN receives inputs from thalamic relay neurons (glutamate, excitatory) and LC/basal forebrain (NE/ACh, modulatory). TRN projects back to relay (GABA, inhibitory). |
| **How to Measure** | **Method 1 (Detection thresholds at different arousal levels):** Measure behavioral detection threshold (contrast at 75% detection) at different times: **Morning (high arousal, A ≈ 0.8):** Low detection threshold (sensitive). **Evening (low arousal, A ≈ 0.5):** Higher detection threshold (less sensitive). **Pre-sleep (very low arousal, A ≈ 0.2):** Highest threshold. Fit arousal-threshold function. θ_0(A) parameters quantify the relationship. **Method 2 (Thalamic gating - fMRI):** fMRI during resting state + oddball (rare stimuli). Measure thalamic response (thalamic relay nucleus, lateral geniculate nucleus for visual). During high arousal: thalamus responds to low-salience stimuli. During low arousal (drowsy): thalamus only responds to high-salience stimuli. Response gain (amplitude at fixed stimulus intensity) → inverse of θ_0. **Method 3 (P3b amplitude modulation):** Oddball task (standards + deviants) at different arousal levels (morning, evening, after sleep deprivation). Measure P3b amplitude as proxy for ignition gain. Higher P3b (lower threshold). Plot P3b vs. estimated arousal. θ_0 ≈ inverse of P3b amplitude. **Method 4 (Thalamic electrophysiology - animal models):** In-vivo single-unit recording from thalamic relay neurons during sleep/wake. Measure threshold for spike-triggered response to sensory input. During wake: lower threshold (~10-20 μV stimulus). During NREM sleep: higher threshold (~50-100 μV stimulus). |
| **Measurement Protocol** | **Behavioral arousal manipulation (1 week protocol):** **Day 1 (high arousal):** Morning test after full sleep. Measure visual detection threshold (contrast discrimination; staircase procedure, ~20 min). Extract 75% detection threshold. **Day 2 (moderate arousal):** Evening test. Same task. **Day 3 (low arousal):** After 20h sleep deprivation (keep awake overnight). Measure threshold at 4 AM. **Day 4 (recovery):** Full sleep night, repeat morning test. Plot thresholds: morning < evening < sleep deprivation (low arousal → high θ_0). Fit linear function: θ_0(A) = a + b*A. Parameters (a, b) quantify arousal modulation. **fMRI thalamic gating (20 min):** fMRI during oddball task in alert state (morning) and drowsy state (evening after partial sleep deprivation). Extract thalamic (LGN for visual) BOLD response to rare deviants. Measure deviant-evoked response amplitude (proxy for threshold). Lower amplitude (higher threshold) in drowsy state. Quantify ratio: BOLD_drowsy / BOLD_alert ≈ θ_0^alert / θ_0^sleep. **P3b modulation (3 sessions):** Oddball at 3 times: 10 AM (alert), 4 PM (moderate), 10 PM (drowsy). Extract P3b amplitude at Pz. Plot amplitude vs. time-of-day (estimated arousal). Fit arousal-dependent function. |
| **Typical Population Values** | **θ_0^alert (high arousal):** ≈ 3-5 nats (conservative threshold). **θ_0^sleep (low arousal):** ≈ 0.5-1.5 nats (sensitive threshold). **Ratio:** θ_0^alert / θ_0^sleep ≈ 2-5× (can vary 2-5 fold with arousal). **Behavioral detection threshold:** Morning ≈ 20% contrast; Evening ≈ 25-30% contrast; Sleep deprived ≈ 40-50% contrast. **P3b amplitude:** Morning ≈ 5-10 μV; Evening ≈ 3-6 μV; Sleep deprived ≈ 1-3 μV. |
| **Pathological Ranges** | **Very low θ_0 (always <1 nat):** Hypervigilance; ignition triggered by minimal evidence (hallucinations, delusions). Seen in: acute psychosis, acute mania, PTSD with hyperarousal, severe anxiety. **Very high θ_0 (always >5 nats):** Hypovigilance; requires extreme evidence to ignite (depression, dissociation). Seen in: severe depression, dissociative disorders, some neurological conditions (thalamic damage). **Flat arousal modulation (θ_0 independent of A):** Loss of arousal regulation. Seen in: brainstem damage, coma, some sleep disorders, advanced dementia. |
| **Individual Differences** | **Genetic:** Catecholamine polymorphisms (COMT Val/Met, MAOA, DAT) → affect baseline threshold tone. **Developmental:** Children: θ_0 lower overall (immature arousal regulation). Adolescence: θ_0 gradually increases. Aging: θ_0 higher (conservative, less responsive to weak signals). **Sleep/Circadian:** Chronotype (morning vs. evening person) shifts arousal modulation (earlier peak time in morning types). **Psychiatric/Neurological:** Depression: elevated θ_0 (conservative threshold). Mania/ADHD: reduced θ_0 (liberal threshold). Schizophrenia: unstable θ_0 (variable ignition). Autism: possibly elevated θ_0 for social/interoceptive signals. |
| **How to Interpret** | **Low θ_0 (easily triggered):** "I need very little evidence to become conscious of things." Adaptive for threat detection, maladaptive for noise filtering (false alarms). **High θ_0 (hard to trigger):** "I need strong evidence to become conscious of things." Adaptive for focus/concentration, maladaptive for threat detection. |
| **Dependencies** | Modulated by: arousal (A), circadian phase, sleep pressure, stress. Correlates with: anxiety/neuroticism, sensory sensitivity, attentional control. |
| **Confounds to Control** | **Circadian phase:** Always test at same clock time, or statistically remove circadian effects. **Sleep history:** Ensure similar sleep (~8 h) before each test. **Arousal state during testing:** Monitor pupil size, heart rate to verify arousal level matches intended. **Stress:** Pre-test stress increases θ_0 temporarily; allow 10 min rest before measurement. |

### Parameter 3b: Metabolic Coupling Strength ($\lambda$)

| Aspect | Details |
| --- | --- |
| **Definition** | Strength of feedback from accumulated signal (S) to threshold: $d\theta/dt = ... + \lambda S(t) + ...$. Implements metabolic cost regulation. |
| **Units** | [1/time]; typical range [0.001, 0.12] s⁻¹. |
| **Meaning** | How strongly does metabolic cost (reflected in accumulated signal S) regulate threshold? **High λ:** Strong feedback; accumulated signals quickly raise threshold (prevent wasteful broadcast). **Low λ:** Weak feedback; threshold insensitive to accumulated signal (ignore metabolic cost). |
| **Biological Implementation** | **Primary:** Feedback from active neural circuits (S ↑) → increased metabolic demand → astrocytes sense lactate/NADH accumulation → feedback to frontier cortex (adenosine, GABA) → raises threshold. **Molecular:** Adenosine (ATP breakdown product) accumulates with neural activity → binds A1 receptors on thalamic relay cells → hyperpolarizes → raises threshold. **Circuit:** Active cortex → high ATP consumption → adenosine accumulation → thalamic inhibition → threshold rise. This is a "metabolic brake." **Long-term:** ATP depletion → loss of membrane potential → seizure threshold lowered (paradoxically). But in normal range, adenosine acts as negative feedback. |
| **How to Measure** | **Method 1 (Indirect: Attention/conscious capacity limits):** Dual-task paradigm. Task A requires conscious attention (e.g., counting red shapes in stream). Task B is simultaneous (e.g., detecting rare sound). Measure interference: When Task A is demanding (high S), Task B detection should drop if λ is high (threshold rises due to Task A metabolic load). If λ is low, Task B detection unaffected. Interference magnitude ≈ λ. **Method 2 (Metabolic imaging):** PET scan with ¹⁸F-FDG (glucose uptake) or ¹¹C-acetate (metabolic rate). Measure brain-wide metabolic rate during conscious task vs. unconscious task. Relationship between conscious signal magnitude and metabolic cost → λ. Task with higher S should have higher metabolic cost if λ > 0. **Method 3 (Pharmacological): Adenosine manipulation:** Administer adenosine (A1 receptor agonist) or caffeine (antagonist). Caffeine blocks adenosine → effectively lowers λ (less metabolic feedback). Predict: Caffeine should increase conscious capacity (can hold more S before threshold rises). Measure conscious capacity (number of items reportable in attentional blink task). Baseline capacity vs. caffeine capacity → difference ≈ λ effect. **Method 4 (Computational modeling):** Fit differential equations to behavioral data (attentional blink, dual-task interference). Estimate λ from best-fitting model. Larger λ → more capacity limitation in dual-task. |
| **Measurement Protocol** | **Dual-task interference (20 min):** **Task A (primary, resource-demanding):** Rapid visual stream with target detection (attend to red shapes; ~20 per min). **Task B (secondary, detection task):** Simultaneous rare audio stimuli (beep; 5 per min). **Baseline:** Single-task (do Task A or B alone). **Dual-task:** Simultaneous A & B. Measure: 1) Task A performance (% detection), 2) Task B performance during Task A. If λ high, Task B performance drops when Task A is demanding. Interference ≈ (SingleTask_B - DualTask_B) / SingleTask_B = Index of λ. Repeat with varying Task A difficulty to measure capacity limits. **Caffeine manipulation (1 week protocol):** Test dual-task capacity on 2 days: **Day 1 (baseline):** Measure dual-task interference (as above). **Day 2 (caffeine):** Consume 200 mg caffeine (2 cups coffee). Wait 30 min. Repeat dual-task. Expected: Improved dual-task performance (reduced interference), indicating lower λ. Compare Day 1 vs Day 2. **Adenosine measurement (experimental):** PET scan during conscious task (high S task) vs. baseline. Measure adenosine occupancy on A1 receptors (¹¹C-CPFPX, adenosine A1 antagonist tracer). Higher occupancy during high-S task → evidence for adenosine feedback. |
| **Typical Population Values** | **From dual-task interference:** Baseline dual-task cost ≈ 20-40% performance loss on Task B when Task A demanding. **From PET metabolic cost:** Conscious visual search ~ 2-3× higher metabolic rate than unconscious (subliminal) search (depends on task difficulty). **From caffeine:** Caffeine reduces dual-task cost by ~15-25% (indicates moderate λ). **Estimated λ:** ~0.032 s⁻¹ (moderate feedback strength). |
| **Pathological Ranges** | **Very low λ (<0.001):** No metabolic constraint; ignition insensitive to cost. Leads to: inability to inhibit consciousness (always broadcasting, high energy waste). Seen in: some ADHD (insufficient inhibition), bipolar mania (racing thoughts). **Very high λ (>0.1):** Extreme metabolic constraint; consciousness suppressed. Leads to: reduced conscious capacity, difficulty maintaining focus. Seen in: severe depression (energy depletion), chronic fatigue syndrome (metabolic limitation). |
| **Individual Differences** | **Genetic:** Mitochondrial genes, COMT, other metabolic genes → affect ATP availability → modulate λ. **Developmental:** Children: λ lower (less metabolic constraint, more conscious capacity). Aging: λ higher (metabolic efficiency declines). **Lifestyle:** Exercise increases mitochondrial density → effectively lowers λ (more ATP available). **Psychiatric/Neurological:** Depression: effectively higher λ (reduced energy). ADHD: possibly lower λ (insufficient cost awareness). **Chronic illness:** Metabolic disorders (diabetes, etc.): altered λ. |
| **How to Interpret** | **High λ (strong feedback):** "I can only think about one thing at a time; trying to do two things makes my brain work harder." Adaptive for focus, maladaptive for multitasking. **Low λ (weak feedback):** "I can keep lots of things in mind; multitasking isn't hard." Adaptive for multitasking, potentially maladaptive (ignores metabolic cost, gets exhausted). |
| **Dependencies** | Related to: attentional capacity (working memory span), conscious resource limitations, brain energetics. Modulated by: metabolic state (fed vs. fasted), exercise, sleep. |
| **Confounds to Control** | **Attention instructions:** Ensure subjects understand to attend to Task A (don't ignore). **Stimulus salience:** Keep Task B stimuli at same salience across conditions (controls for baseline Task B threshold). **Circadian timing:** Test at same time of day (metabolism varies circadianly). **Metabolic state:** Test in fed state (controls for blood glucose). **Caffeine usage:** Document baseline caffeine consumption; controls between subjects will differ. |

## PART 4: IGNITION SHARPNESS & GAIN PARAMETERS

### Parameter 4a: Logistic Steepness ($\alpha$)

| Aspect | Details |
| --- | --- |
| **Definition** | Steepness of logistic ignition function: $B = \sigma(\alpha(S - \theta)) = 1/(1 + \exp(-\alpha(S-\theta)))$. Controls graded vs. all-or-none ignition. |
| **Units** | [1/surprise units] = [dimensionless/nats]; typical range [0.5, 12.0] nats⁻¹. |
| **Meaning** | **High α:** Sharp transition (nearly all-or-none ignition; small changes in S near threshold cause large changes in B). **Low α:** Gradual transition (graded ignition; broad range of S values produce intermediate probabilities). |
| **Biological Implementation** | **Primary:** Stochastic noise in neural circuits (ion channel noise, synaptic variability) sets the transition steepness. **Equation:** α = 1/σ_noise, where σ_noise is effective noise. Low noise → high α (sharp). High noise → low α (gradual). **Molecular sources of noise:** Stochastic ion channel opening (Poisson), neurotransmitter release variability (binomial), background synaptic bombardment. **Circuit:** Thalamic relay neurons must reach threshold to fire. Variability in subthreshold inputs → variability in threshold crossing → gradual transition. More synchronized/structured input → sharper transition. |
| **How to Measure** | **Method 1 (Psychometric function slope):** Detection task with varying stimulus strength (contrast: 5%, 10%, 25%, 50%, 100%). Measure detection probability P(detect) at each contrast. Plot psychometric curve: P(detect) vs. log(contrast). Fit logistic: P = 1 / (1 + exp(-α*(log(contrast) - log(threshold)))). Slope at 50% point ≈ α/2.3 (convert from log scale). Steeper slope → larger α. **Method 2 (ROC curve):** Signal detection task. Present noise-alone vs. signal+noise trials. Vary signal strength. Plot detection sensitivity (d-prime) vs. criterion. Calculate slope of ROC curve at operating point. Slope ≈ α. **Method 3 (Response latency distribution):** Detection task with reaction time measured. Collect RT distribution at threshold signal strength. Analyze RT distribution: **Sharp threshold (high α):** Bimodal RT (either quick "hit" or slow "miss"). **Gradual threshold (low α):** Unimodal RT (intermediate values common). Characterize bimodality → α (higher bimodality → higher α). **Method 4 (Neural response variability):** Single-unit recording from thalamic relay neuron. Present stimulus at varying strengths. Measure whether neuron fires (spike/no-spike) on each trial. At threshold stimulus, compute probability of firing across trials. Plot P(fire) vs. stimulus strength. Fit logistic → α. **Method 5 (EEG amplitude distribution):** Extract P3b amplitudes at near-threshold stimulus strength. If α high: bimodal distribution (large P3b on "seen" trials, small on "unseen"). If α low: unimodal distribution. Fit mixture model to amplitude distribution. |
| **Measurement Protocol** | **Psychometric function (20 min):** Visual detection task. Present Gabor patches at varying contrasts (5%, 10%, 25%, 50%, 100%). 20 trials per contrast, randomized. Subject reports "seen" or "unseen." Compute P(seen) for each contrast. Fit cumulative Gaussian (or logistic): P(seen) = Φ((log(contrast) - log(threshold))/σ). σ parameter relates to α: α ≈ 1/(σ*ln(10)) (converts from log to linear scale). Plot logistic curve. **ROC analysis (15 min):** Signal detection task. Variable contrast stimulus (50 levels: 1-100%) or signal+noise (SNR: 0-10 dB). 10 trials per level. Subject rates confidence in 5-point scale (certainly unseen, probably unseen, neutral, probably seen, certainly seen). Plot true positive rate (hits) vs. false positive rate (false alarms) for each confidence level. Fit ROC. Slope at iso-discrimination point ≈ α. **Latency variability (20 min):** Reaction time detection task. Stimulus at near-threshold contrast (estimated from preliminary staircase procedure). 100 trials. Measure RT... Analyze RT distribution: **Bimodality test (using dip test):** If multimodal (two peaks), bimodality exists, suggesting high α. If unimodal, suggests low α. Quantify bimodality index... **EEG amplitude distribution (20 min):** Threshold detection task with simultaneous EEG. Extract P3b amplitudes on each trial. Fit mixture model: P(amplitude) = pf* N(μ_seen, σ_seen) + (1-pf) * N(μ_unseen, σ_unseen), where pf = proportion of "seen" trials (≈50% for threshold stimulus). Separation between μ_seen and μ_unseen indicates α (larger separation → larger effective α). |
| **Typical Population Values** | **From psychometric function:** Slope parameter σ ≈ 0.1-0.3 (log units); converts to α ≈ 1-3 nats⁻¹. **From ROC:** Slope ≈ 1-3. **From latency:** For high α (bimodal RT): RT_seen ≈ 300-400 ms, RT_unseen ≈ 600-800 ms. For low α (unimodal RT): RT distributed 300-800 ms. **From EEG:** P3b separation (seen vs. unseen): μ_seen - μ_unseen ≈ 2-4 μV (well-separated, high α) vs. ≈ 0.5-1 μV (overlapping, low α). |
| **Pathological Ranges** | **Very high α (>5):** All-or-none ignition (abrupt access). Seen in: acute psychosis (abrupt thought insertions), seizures (abrupt loss of consciousness). **Very low α (<0.5):** Extremely gradual threshold; consciousness very probabilistic. Seen in: severe intoxication (unstable consciousness), some sleep disorders (shallow sleep). |
| **Individual Differences** | **Genetic:** Noise-related genes (ion channel genes, synaptic vesicle genes) → affect neural stochasticity → modulate α. **Developmental:** Children: α lower (more noise, less mature circuits). Aging: α decreases (~5-10%/decade) (increased noise in aging brain). **Lifestyle:** Meditation/training: α may increase (refined circuits, less noise). Sleep deprivation: α decreases (noise increases). **Psychiatric/Neurological:** Schizophrenia: α variable/reduced (noisy perception). Autism: possibly higher α (more structured processing). Depression: possibly lower α (flat affect). |
| **How to Interpret** | **High α (sharp):** "Once I reach a threshold, I become conscious of it pretty suddenly." Adaptive for clear conscious boundaries, maladaptive for false positives (all-or-none errors). **Low α (gradual):** "I gradually become conscious of weak signals over time." Adaptive for noise robustness (graceful degradation), maladaptive for missed signals (threshold ambiguity). |
| **Dependencies** | Related to: neural circuit maturity, noisiness of neural processing, consciousness "clarity." Modulated by: sleep, intoxication, stress. |
| **Confounds to Control** | **Stimulus clarity:** Ensure stimuli are clearly perceptible at high contrast (controls for baseline sensitivity). **Attention:** Explicit instruction to attend maximally (controls for attention confound). **Decision criterion:** Clarify instruction ("report even if unsure" vs. "only report if confident") to control for criterion shifts that mimic α changes. |

### Parameter 4b: Somatic Marker Sensitivity ($\beta_M$)

| Aspect | Details |
| --- | --- |
| **Definition** | Sensitivity of somatic marker target to interoceptive errors: $M^*(\varepsilon^i) = \tanh(\beta_M \varepsilon^i)$. |
| **Units** | [dimensionless]; typical range [0.08, 11.0]. |
| **Meaning** | How much does body state mismatch (ε^i) contribute to predicted homeostatic cost (M)? **High β_M:** Small body state changes produce large cost predictions (body signals are highly salient). **Low β_M:** Large body state changes produce small cost predictions (body signals are dampened). |
| **Biological Implementation** | **Primary:** Insular cortex (posterior insula: primary somatosensory representation; anterior insula: emotional/interoceptive integration) translates body state signals (ε^i) to emotional valuation (M). **Molecular:** BDNF (brain-derived neurotrophic factor) modulates insula synaptic strength (higher BDNF → higher β_M). Opioids (endogenous and exogenous) reduce β_M (dampening pain/threat). **Circuit:** Vagus nerve → nucleus tractus solitarius → posterior insula (sensory) → anterior insula (emotion); anterior insula → vmPFC (valuation). Synaptic strength in these pathways determines β_M. |
| **How to Measure** | **Method 1 (Heart rate stimulus-response):** Measure actual heart rate variability (HRV) changes induced by experimental manipulation (orthostatic challenge, Valsalva maneuver, cold pressor test, mental arithmetic stress). Measure vmPFC/insula response (fMRI BOLD) to the same manipulation. Correlate ε^i (measured HRV change) with M (vmPFC BOLD response). Slope of relationship ≈ β_M. Steeper slope → higher β_M. **Method 2 (Pain interoceptive mapping):** Administer noxious stimuli (heat, cold, pressure) at various intensities. Measure: 1) Actual nociception (skin conductance, heart rate, cortisol). 2) vmPFC response (fMRI). 3) Self-reported pain. Relate nociception (ε^i) to vmPFC response (M). Slope ≈ β_M. **Method 3 (Interoceptive accuracy for pain):** Heartbeat detection task during pain (mild shock, heat). Correlate heartbeat detection accuracy with pain sensitivity/catastrophizing scores. Better detection → higher β_M (more attention to bodily signals). **Method 4 (Insula-vmPFC connectivity):** rsfMRI or task fMRI during pain/threat. Measure functional connectivity between insula and vmPFC. Stronger connectivity → higher effective β_M. Quantify as Pearson r between timeseries. |
| **Measurement Protocol** | **Orthostatic challenge + fMRI (15 min):** Subject lies in MRI scanner. fMRI active. Baseline (lying flat, 5 min). Challenge 1 (tilt head to 10° upright, 3 min) → induces orthostatic stress, heart rate increase (~10 bpm). Return to baseline (2 min). Challenge 2 (tilt to 20° upright, 3 min) → stronger stress, heart rate increase (~20 bpm). Return to baseline. Measure: Heart rate change (ε^i) at each challenge. Extract vmPFC BOLD change (from pre-defined ROI: -4, 52, -8). Correlate ε^i with BOLD. Slope ≈ β_M. **Pain intensity task (20 min):** Present heat stimuli at 5 intensities (35°C baseline, 43°C, 45°C, 47°C, 49°C). 10 trials per temperature, randomized. Simultaneously measure: 1) Skin conductance (physiological stress response = ε^i). 2) Insula/vmPFC BOLD. 3) Self-report pain rating. Correlate physiological stress response (ε^i) with vmPFC activation (M). Linear regression: M = β_0 + β_M*ε^i. β_M coefficient ≈ somatic marker sensitivity. **Connectivity analysis (10 min):** Resting-state fMRI. Extract time series from insula seed (-36, 12, 6) and vmPFC (-4, 52, -8). Compute Pearson correlation between time series. Higher correlation (r > 0.3) → higher effective β_M. Compare across individuals. |
| **Typical Population Values** | **From pain task:** β_M ≈ 2.05 (pain-induced physiological change produces proportional vmPFC response). **From connectivity:** Insula-vmPFC r ≈ 0.2-0.5 (moderate to strong connectivity). **From heartbeat detection during pain:** Correlation (detection accuracy vs. pain catastrophizing) r ≈ 0.3-0.5 (moderate; higher in anxiety). |
| **Pathological Ranges** | **Very low β_M (<0.5):** Interoceptive dampening; body signals don't produce emotional response. Seen in: dissociation, autism spectrum (reduced body awareness), depression (anhedonia), some opioid use. **Very high β_M (>5):** Interoceptive amplification; minor body changes produce exaggerated emotional responses. Seen in: anxiety disorders, panic disorder, PTSD, chronic pain conditions. |
| **Individual Differences** | **Genetic:** BDNF Val66Met: Val carriers → higher BDNF → higher β_M. Opioid receptor genes (OPRM1): variations affect β_M via endogenous opioids. **Developmental:** Early adversity/trauma → reduced β_M (protective dissociation becomes habitual). **Experience:** Yoga/somatic therapy → increases β_M (learns to integrate body signals). **Psychiatric history:** Trauma → reduced β_M (dissociation). Anxiety → elevated β_M (threat-related amplification). **Sex differences:** Females typically higher β_M than males (~20-30% higher interoceptive sensitivity). |
| **How to Interpret** | **High β_M:** "Small changes in my body feel very significant emotionally." Adaptive for health monitoring, maladaptive for anxiety (somatic amplification). **Low β_M:** "My body signals don't affect my emotions much." Adaptive for emotional regulation, maladaptive for health awareness. |
| **Dependencies** | Related to: anxiety sensitivity, pain catastrophizing, interoceptive awareness. Modulated by: trauma history, emotional state, medications (opioids reduce it). |
| **Confounds to Control** | **Stimulus intensity:** Use calibrated stimuli (same objective intensity across subjects). **Attention:** Explicit instruction to monitor body (vs. distract from body). **Expectations:** Use surprise (unexpected temperature changes) to isolate reflexive response from anticipatory response. |

## PART 5: DYNAMIC PARAMETERS

### Parameter 5a: Homeostatic Recovery Rate ($\gamma$)

| Aspect | Details |
| --- | --- |
| **Definition** | Rate constant for post-ignition threshold recovery: $\theta(t) = \theta_0 (1 - e^{-\gamma t})$ |
| **Units** | [1/second]; typical range [0.015, 0.085] |
| **Meaning** | How quickly does threshold return to baseline after ignition? High γ = fast recovery (quick reset). Low γ = slow recovery (prolonged refractory). |
| **Biological Implementation** | **Primary:** Autonomic homeostatic feedback restoring baseline state. **Molecular:** Adenosine clearance, GABA_A receptor kinetics. **Circuit:** Thalamocortical feedback loops restore baseline excitability. |
| **How to Measure** | **Method 1 (Threshold recovery):** Measure detection threshold before and after ignition events. Fit exponential recovery. **Method 2 (P3b amplitude):** P3b amplitude vs. inter-trial interval. Recovery rate → γ. |
| **Measurement Protocol** | **Threshold recovery (integrated):** Detection task with trials at varying ISIs. Fit: θ(t) = θ₀(1 - e^(-γt)). γ ≈ recovery rate constant. |
| **Typical Population Values** | γ ≈ 0.048. **By group:** Healthy: γ ≈ 0.05. ADHD: γ ≈ 0.07 (faster recovery). Depression: γ ≈ 0.03 (slower). |
| **Pathological Ranges** | **Very low γ (<0.015):** Very slow recovery; prolonged refractory (severe depression, fatigue). **Very high γ (>0.085):** Very fast recovery; no refractory (ADHD, mania). |
| **Individual Differences** | **Genetic:** Adenosine receptor genes affect clearance rate. **Developmental:** Children: γ higher. Adults: γ moderate. **Lifestyle:** Exercise may increase γ. |
| **How to Interpret** | **High γ:** "I bounce back quickly after mental effort." Adaptive for multitasking. **Low γ:** "I need time to recover after focusing." Adaptive for sustained attention. |
| **Dependencies** | Related to: autonomic homeostasis, metabolic state. Modulated by: fatigue, stress. |
| **Confounds to Control** | **Fatigue:** Test at consistent energy level. **Stress:** Control stress exposure. **Timing:** Precise ISI control. |

### Parameter 5b: Post-Ignition Elevation ($\delta$)

| Aspect | Details |
| --- | --- |
| **Definition** | Immediate threshold elevation after ignition: $\theta_{\text{post}} = \theta_0 (1 + \delta)$. Fractional increase. |
| **Units** | [dimensionless]; typical range [0.08, 0.35] |
| **Meaning** | How much does threshold rise immediately after ignition? High δ = large elevation (strong refractory). Low δ = minimal elevation (weak refractory). |
| **Biological Implementation** | **Primary:** Refractory period from neuromodulator depletion and receptor desensitization. **Molecular:** Adenosine accumulation, GABA_B receptor activation. **Circuit:** Thalamic inhibition from accumulated metabolic cost. |
| **How to Measure** | **Method 1 (Threshold measurement):** Measure threshold immediately (<500ms) after detection vs. baseline. δ = (θ_post - θ₀)/θ₀. **Method 2 (RT distribution):** Bimodal RT distribution indicates refractory. Separation → δ. |
| **Measurement Protocol** | **Threshold elevation (integrated):** Detection task with rapid successive trials. Measure threshold on trial N vs. trial N+1. Fit: θ(t) = θ₀(1 + δe^(-γt)). |
| **Typical Population Values** | δ ≈ 0.19. **By group:** Healthy: δ ≈ 0.2. ADHD: δ ≈ 0.08 (minimal elevation). Depression: δ ≈ 0.35 (large elevation). |
| **Pathological Ranges** | **Very low δ (<0.08):** No refractory; rapid successive ignitions (ADHD, mania). **Very high δ (>0.35):** Excessive refractory; slow processing (depression, fatigue). |
| **Individual Differences** | **Genetic:** Metabolic genes affect depletion rate. **Developmental:** Children: δ lower. Adults: δ moderate. **Lifestyle:** Fatigue increases δ. |
| **How to Interpret** | **High δ:** "I get stuck after focusing; hard to switch tasks." Adaptive for sustained attention. **Low δ:** "I can switch tasks rapidly." Adaptive for multitasking. |
| **Dependencies** | Related to: metabolic cost, neuromodulator depletion. Modulated by: fatigue, arousal. |
| **Confounds to Control** | **Fatigue:** Test at consistent energy level. **Timing:** Precise post-ignition measurement. **Stress:** Control stress exposure. |

## PART 6: IMPLEMENTATION RECOMMENDATIONS

### For Research Teams (Multi-Site Studies)

1. **Standardize measurement protocols** across sites (use template above)
2. **Quality control:** Repeat 10% of measurements (test-retest reliability; ICC > 0.7)
3. **Data sharing:** Upload anonymized data to open repository (OSF, BioStudies) for transparency
4. **Cross-validation:** Fit APGI model to behavioral data; test predictions on held-out data

### For Individual Researchers

1. **Start with easy parameters** (Π^e, τ_S via psychophysics; pupil via simple eye-tracking)
2. **Add neuroimaging gradually** (P3b via EEG if available; fMRI if needing deeper investigation)
3. **Focus on individual differences:** Which parameters vary most across your sample?
4. **Link to clinical outcomes:** How do parameter values predict symptoms, treatment response, prognosis?

### For Clinicians

1. **Simple screening battery** (~30 min):
   - Heartbeat detection (Π^i)
   - Pupil response (τ_A)
   - P3b (θ_0)
   - Dual-task interference (λ)

2. **Interpret as biomarkers:**
   - High Π^i + elevated τ_M → heightened threat sensitivity (anxiety)
   - Low Π^i + low β_M → interoceptive insensitivity (dissociation, depression)
   - Elevated λ → metabolic constraint (depression, fatigue)

3. **Monitor treatment response:** Re-test parameters after intervention; expect normalization correlates with symptom improvement
