# Codebooks Repository

This directory contains codebooks and data dictionaries for the datasets.

## Expected File Formats:
- Markdown files with variable descriptions
- CSV files with coding schemes
- Text files with data format specifications

## Codebook Contents:
Each codebook should document:
- Variable names and descriptions
- Data types and units
- Valid value ranges
- Missing data codes
- Coding schemes for categorical variables
- Data collection procedures

## Example Codebook Entry:
```
Variable: eeg_fz
Description: EEG amplitude at Fz electrode
Unit: microvolts (µV)
Type: continuous
Range: -100 to +100 µV
Missing: NA
```

## Note:
Codebooks ensure proper interpretation and analysis of the data variables.
